# 术语表

## 一、核心概念

| 中文 | 英文 | 缩写 | 说明 |
|------|------|------|------|
| 超声波流量计 | Ultrasonic Flow Meter | UFM | 利用超声传播时间差测量流量的仪表 |
| 声道 | Chord / Acoustic Path | — | 超声波在管道中穿行的路径 |
| 顺流传播时间 | Upstream Travel Time | $t_+$ | 超声波顺流向传播的时间 |
| 逆流传播时间 | Downstream Travel Time | $t_-$ | 超声波逆流向传播的时间 |
| 传播时间差 | Transit Time Difference | $\Delta t$ | $\Delta t = t_- - t_+$ |
| 声道长度 | Chord Length | $L$ | 超声波在流体中的传播路径长度 |
| 声道安装角 | Chord Angle | $\theta$ | 声道与管道轴线的夹角 |
| 投影因子 | Projection Factor | $\cos\theta$ | 流速在声道方向投影，本题为 $1/\sqrt{2}$ |
| 面平均流速 | Cross-sectional Mean Velocity | $\bar{v}$ | 管道截面积加权的平均流速 |
| 体积流量 | Volume Flow Rate | $Q$ | 单位时间通过截面的流体体积 |
| 窗口 | Window | — | 连续测试过程切分的计算单元 |
| 等效观测量 | Equivalent Observation | $c_j$ | 五声道的窗口级归一化标量，可理解为各声道对速度剖面的等效观测 |
| 扰流状态 | Disturbance Condition | D0–D8 | D0 无扰流，D1–D8 八种扰流 |
| 雷诺数 | Reynolds Number | $Re$ | 表征流动状态的量纲为一数，影响湍流速度剖面形状 |

## 四、多声道积分

| 中文 | 英文 | 说明 |
|------|------|------|
| 高斯-雅可比积分 | Gauss-Jacobi Quadrature | 圆管截面数值积分，权重函数 $(1-t^2)^{1/2}$ |
| 高斯-勒让德积分 | Gauss-Legendre Quadrature | 均匀权重数值积分 |
| 拉格朗日积分 | Lagrange Integration | 插值型积分 |
| 五声道加权积分 | 5-Chord Weighted Integration | $V = A \cdot T \cdot \sum w_j c_j$ |
| 声道0/4 | Chord 0/4 (Edge) | 边缘声道，$\vert y\vert \approx 0.727R$ |
| 声道1/3 | Chord 1/3 (Inner) | 内侧声道，$\vert y\vert \approx 0.266R$ |
| 声道2 | Chord 2 (Center) | 中心声道，$y = 0$ |

## 五、基础积分方案

| 缩写 | 全称 | 特征 |
|------|------|------|
| Phys6 | Physical 6-path Integration | 工程原始系数，五项和 ≈ 0.993（不归一） |
| OWICS | OWICS Integration | 归一化声道积分，与 Gauss-Jacobi 一致 |
| Lagrange | Lagrange-type Integration | Lagrange 插值型，均值误差最接近零 |
| Equal Weight | Equal Weight Integration | 五声道简单算术平均 |

## 七、扰流相关概念

| 中文 | 英文 | 说明 |
|------|------|------|
| 扰流件 | Flow Disturber | 安装在管道中的扰流元件（弯头、阀门、缩径等） |
| 扰流检测 | Disturbance Detection | 在线判断当前窗口是否受扰流影响 |
| 扰流分类 | Disturbance Classification | 将扰流状态归为若干剖面畸变类型 |
| 扰流补偿 | Disturbance Compensation | 根据扰流类型修正流量估计的系统偏差 |
| 双层识别 | Two-Layer Recognition | 第1层检测有无扰流，第2层判别扰流类型 |
| 在线规则 | Online Rule | 不依赖人工扰流编号、仅用在线特征的判定逻辑 |
| 剖面畸变 | Profile Distortion | 扰流导致的截面速度分布偏离无扰流理想形状 |
| 旋流 | Swirl Flow | 流体绕管道轴线的旋转运动 |
| 非对称流 | Asymmetric Flow | 管道截面上下或左右速度分布不对称 |

## 八、评价指标

| 中文 | 英文 | 符号 | 公式/定义 |
|------|------|------|----------|
| 相对误差 | Relative Error | $e_i$ | $e_i = (\hat{V}_i - V_i^{\text{std}}) / V_i^{\text{std}} \times 100\%$ |
| 平均绝对误差 | Mean Absolute Error | MAE | $\frac{1}{n}\sum \vert e_i\vert$ |
| 均值误差 | Mean Error / Bias | — | $\frac{1}{n}\sum e_i$ |
| 组均值误差 | Group Mean Error | $\bar{e}_g$ | $\frac{1}{n_g}\sum_{k=1}^{n_g} e_{g,k}$ |
| 组内重复性 | Group Repeatability SD | $SD_g$ | $\sqrt{\frac{1}{n_g-1}\sum(e_{g,k} - \bar{e}_g)^2}$ |
| 组达标 | Group Pass | — | $\vert\bar{e}_g\vert \le 0.2\%$ 且 $SD_g \le 0.040\%$ |
| 线性度等效量 | Normalized Linearity | $u_{\text{nor},L}$ | $\sqrt{\frac{1}{m-1}\sum_{p=1}^{m}(\bar{e}_p^{(0)})^2}$ |
| 重复性等效量 | Normalized Repeatability | $u_{\text{nor},r}$ | $\max_g SD_g$ |
| 扰流综合指标 | Normalized Disturbance | $u_{\text{nor},d}$ | $\sqrt{u_{\text{nor},d,c}^2 + u_{\text{nor},d,r}^2}$ |

## 九、模型架构术语

| 中文 | 英文 | 说明 |
|------|------|------|
| 基线积分 | Baseline Integration | 固定权重多声道积分（Lagrange） |
| 流量点修正 | Flow-Point Correction | 按流量点查表修正系统偏差 |
| 剖面修正 | Profile Correction | 利用剖面特征修正窗口级偏差 |
| 有效性门 | Validity Gate | 基于动态特征判断窗口是否有效 |
| 消融实验 | Ablation Study | 逐模块移除以量化各模块贡献 |
| 级联架构 | Cascade Architecture | 按"基线→修正→检测→补偿"顺序的模块串联 |
| 冷启动 | Cold Start | 系统刚启动时零点信息不足的边界工况 |
| 留出验证 | Leave-One-Out Validation | 留出一天/一组做测试，其余做训练的交叉验证 |
| Bootstrap | Bootstrap | 重采样估计小样本参数的不确定性区间 |
| 层次聚类 | Hierarchical Clustering | Ward 法 + 轮廓系数确定扰流聚类数 |
| k-近邻 | k-NN | k=3，到聚类中心最近邻的扰流分类方法 |
| Cohen's d | Cohen's d | 效应量指标，衡量两类均值差异的标准化幅度 |

# 问题3：有扰流条件下的剖面识别与补偿 — 建模分析

## 一、问题分析

### 1.1 问题背景

149 个有扰流窗口覆盖 D1–D8 八种扰流状态，分布在 20260608–20260615 共 8 天。扰流件导致速度剖面畸变（非对称流、旋流、偏流），使固定权重积分产生系统性偏差。需要识别扰流特征规律并建立补偿模型。

### 1.2 问题重述（四问）

| 子问题 | 核心任务 |
|--------|---------|
| (1) 特征选择 | 哪些特征对区分无扰流和有扰流最有用？ |
| (2) 扰流聚类 | D1–D8 可否归并为若干类？给出聚类依据 |
| (3) 在线识别 | 不读取 `disturbance_id`，构造扰流状态在线识别规则 |
| (4) 流量补偿 | 扰流下是否需对不同流量点采用不同补偿？ |

### 1.3 核心约束

- 离线分析可使用 `disturbance_id` 研究规律
- **在线规则不得依赖人工输入的扰流编号**
- 补偿规则和参数须在测试前固定

### 1.4 问题类型

特征工程 + 统计判别 + 聚类分析 + 分段补偿。核心是挖掘 profile/AB差异/dynamic/zero 四类特征对扰流的响应规律。

## 二、子问题(1)：扰流敏感特征识别

### 2.1 候选特征池

| 特征类别 | 具体变量 | 物理含义 |
|---------|---------|---------|
| 声道等效观测 | chord0–chord4 | 五声道等效流速贡献 |
| A/B方向差异 | ab0–ab4 | 同声道双方向不对称度 |
| 剖面特征 | profile_top_bottom, profile_center_all, profile_edge_inner, profile_inner_skew | 截面速度分布形状 |
| A/B汇总 | profile_ab_abs, profile_swirl | 方向差异总量 + 旋流代理 |
| 启停动态 | dyn_first_0p1_s, dyn_tail_0p1_s, dyn_start_over_plateau, dyn_end_over_plateau, dyn_plateau_cv, dyn_active_eq_s | 窗口稳定性 |
| 零点状态 | zero_rate_med, zero_rate_mad, zero_age_s | 零漂基线 |

### 2.2 判别力评估方法

对每个特征 $x$，定义两类区分度指标：

**效应量（Cohen's d）**：

$$d = \frac{\bar{x}_D - \bar{x}_{D0}}{\sqrt{(s_D^2 + s_{D0}^2)/2}}$$

其中 $\bar{x}_D, s_D$ 为有扰流（D1–D8）的均值和标准差；$\bar{x}_{D0}, s_{D0}$ 为无扰流（D0）。

**AUC（ROC曲线下面积）**：以特征值作为判别分数，计算区分 D0 vs D1–D8 的 AUC。

### 2.3 数据分析结果

基于 159 窗口实际数据：

| 特征 | D0 均值±SD | D1–D8 均值±SD | Cohen's d | 判别力 |
|------|-----------|--------------|-----------|--------|
| **profile_ab_abs** | 0.0115±0.0059 | 0.2101±0.0060 | **33.1** | ★★★★★ |
| **profile_swirl** | -0.0060±0.0037 | 0.1230±0.0028 | **39.6** | ★★★★★ |
| profile_top_bottom | -0.0118±0.0039 | -0.0293±0.0508 | 0.5 | ★★ |
| profile_edge_inner | 0.9258±0.0024 | 0.9503±0.0108 | 3.0 | ★★★★ |
| profile_center_all | 1.0387±0.0016 | 1.0256±0.0060 | 2.8 | ★★★★ |
| zero_rate_med | 0.8351±0.0633 | 1.7475±0.6686 | 1.9 | ★★★ |
| zero_rate_mad | 0.1227±0.0328 | 0.2837±0.3105 | 0.7 | ★★ |
| zero_age_s | 272.8±264.0 | 865.3±606.6 | 1.3 | ★★ |

### 2.4 结论

**最强判别特征**：`profile_ab_abs` 和 `profile_swirl`。两者在 D0 中接近零，在 D1–D8 中显著偏离零，物理含义明确（扰流件引入方向不对称和旋流），且变异系数极低（D1–D8 中 CV ≈ 3%），适合作为在线检测的核心指标。

## 三、子问题(2)：扰流状态聚类

### 3.1 聚类特征

以五声道等效观测量（chord0–chord4）归一化剖面 + A/B 差异 + 剖面特征构建特征向量：

$$z = [\tilde{c}_0,\, \tilde{c}_1,\, \tilde{c}_2,\, \tilde{c}_3,\, \tilde{c}_4,\; \mathrm{ab}_0, \ldots, \mathrm{ab}_4,\; p_{\mathrm{abs}},\; p_{\mathrm{swirl}}]$$
其中 $p_{\mathrm{abs}} = \mathrm{profile\_ab\_abs}$，$p_{\mathrm{swirl}} = \mathrm{profile\_swirl}$

其中 $\tilde{c}_j = c_j / \sum_k c_k$ 为归一化声道贡献（消除流量幅度影响，仅保留剖面形状）。

### 3.2 方法：层次聚类 + 轮廓系数

**步骤**：
1. 对每个 D1–D8 扰流状态，取该状态下所有窗口的特征均值作为该扰流的代表向量
2. 计算 8×8 欧氏距离矩阵
3. 用 Ward 法做层次聚类，以轮廓系数确定最优聚类数

**判别依据**：
- 类内紧致：同类的扰流状态剖面畸变模式相似
- 类间分离：不同类的扰流状态剖面畸变模式有本质差异
- 物理可解释：每类对应一种扰流机理（如弯头效应、半板阻挡、缩径等）

### 3.3 预期结果

预期 8 种扰流可归并为 2–4 类：

| 聚类 | 可能机理 | 剖面特征 |
|------|---------|---------|
| 类型A | 对称收缩/扩张 | chord1–chord3 均匀偏离 |
| 类型B | 单侧扰流（弯头） | chord0/chord1 与 chord3/chord4 不对称 |
| 类型C | 旋流主导 | profile_swirl 大，profile_ab_abs 大 |
| 类型D | 混合型 | 多特征同时异常 |

## 四、子问题(3)：扰流在线识别规则

### 4.1 问题约束

在线运行时不能读取 `disturbance_id`，必须仅用窗口级在线可计算特征判断是否存在扰流及其类型。

### 4.2 双层识别架构

```
窗口特征
    │
    ▼
┌──────────────────┐
│ 第1层：扰流检测   │  ← 二分类：D0 vs D1–D8
│ (profile_ab_abs  │
│  + profile_swirl) │
└──────┬───────────┘
       │ 判定为扰流
       ▼
┌──────────────────┐
│ 第2层：扰流分类   │  ← 多分类：A/B/C/D 型
│ (chord剖面 +     │
│  AB差异模式)     │
└──────┬───────────┘
       │
       ▼
   对应补偿策略
```

### 4.3 第1层：扰流检测规则

**双指标联合判定**：

$$\mathrm{is\_disturbed} = \begin{cases} 1, & |p_{\mathrm{abs}}| > \tau_1 \;\mathrm{OR}\; |p_{\mathrm{swirl}}| > \tau_2 \\[4pt] 0, & \text{otherwise} \end{cases}$$

阈值设定：以 D0 的 max 值为基准加 3σ 余量：

$$\tau_1 = \max_{i \in D0}|p_{\mathrm{abs},i}| + 3\sigma_{D0} = 0.0177 + 3 \times 0.0059 = 0.035$$

$$\tau_2 = \max_{i \in D0}|p_{\mathrm{swirl},i}| + 3\sigma_{D0} = 0.0097 + 3 \times 0.0037 = 0.021$$

**可解释性**：
- `profile_ab_abs`（即 $p_{\mathrm{abs}}$）：A/B 方向差异绝对量。无扰流时声道双向对称，此值应接近零；扰流件破坏对称性
- `profile_swirl`（即 $p_{\mathrm{swirl}}$）：旋流代理。无扰流时截面无系统性旋转，此值接近零；扰流件（如弯头、旋流发生器）产生可测旋流

### 4.4 第2层：扰流类型判别

以归一化五声道剖面 $\tilde{c} = [\tilde{c}_0, \ldots, \tilde{c}_4]$ 和归一化 AB 差异 $\widetilde{ab} = [\widetilde{ab}_0, \ldots, \widetilde{ab}_4]$ 拼接为 10 维特征向量。

**方法**：k-近邻（k=3）到聚类中心。用子问题(2)的聚类结果标记各窗口所属扰流类型，新窗口特征与各类中心最近邻归类。

**避免过拟合**：仅用 4 个聚类中心（而非 8 个扰流编号），大幅降低分类复杂度。

## 五、子问题(4)：分流量点扰流补偿

### 5.1 问题分析

扰流导致的剖面畸变在不同流量下表现不同：
- 低流量（20–30）：流速低，惯性效应弱，扰流影响更局部
- 高流量（60–80）：流速高，湍流混合强，扰流影响更分散

需要判断是否需分流量点补偿。

### 5.2 分析框架

对每种扰流类型（聚类后的类别），在 7 个流量点上分别计算扰流引入的偏差：

$$\Delta_{p}^{(c)} = \bar{e}_{p,\mathrm{dist}}^{(c)} - \bar{e}_{p,\mathrm{ref}}$$

若 $\Delta_{p}^{(c)}$ 在不同 $p$ 之间差异显著（ANOVA p < 0.05），则该扰流类型需分流量点补偿。

### 5.3 补偿模型

**形式一：加性补偿**（推荐）

$$V_{\mathrm{final}} = V_{\mathrm{base}} \cdot \bigl(1 + \beta_p + \delta_{p}^{(c)}\bigr)$$

其中：
- $\beta_p$：问题2的无扰流流量点修正
- $\delta_{p}^{(c)}$：扰流类型 $c$ 在流量点 $p$ 的额外修正

$\delta_{p}^{(c)}$ 由该扰流类型内各流量点的平均偏差估计：

$$\delta_{p}^{(c)} = \frac{1}{N_{p}^{(c)}} \sum_{i \in \Omega_{p}^{(c)}} \frac{V_i^{\mathrm{std}} - V_i^{\mathrm{base}}}{V_i^{\mathrm{base}}} - \beta_p$$

**形式二：乘性补偿**（备选）

$$V_{\mathrm{final}} = V_{\mathrm{base}} \cdot (1 + \beta_p) \cdot (1 + \gamma^{(c)})$$

仅一个全局扰流因子 $\gamma^{(c)}$，适用于各流量点偏差一致的扰流类型。

### 5.4 补偿策略选择

通过 ANOVA 检验每种扰流类型的 $\Delta_{p}^{(c)}$ 跨流量点是否一致：

| ANOVA 结果 | 策略 |
|-----------|------|
| p ≥ 0.05（无显著差异） | 全局乘性补偿 $\gamma^{(c)}$ |
| p < 0.05（有显著差异） | 分流量点加性补偿 $\delta_p^{(c)}$ |

## 八、注意事项

1. D0 仅 10 窗口，效应量估计有不确定性——应汇报 Cohen's d 的置信区间
2. 扰流识别规则的阈值需在论文中给出完整的物理推导，而非纯数据驱动
3. 补偿参数不得使用 `standard_volume_m3` 按窗口调优——必须按组（扰流类型×流量点）聚合
4. 最终模型中的扰流检测和分类规则必须仅依赖在线可计算特征
5. `profile_ab_abs` 和 `profile_swirl` 的判别力极强（d > 30），论文中应强调这一物理发现

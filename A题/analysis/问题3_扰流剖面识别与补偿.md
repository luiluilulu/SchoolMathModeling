# 问题3：有扰流条件下的剖面识别与补偿 — 建模分析

## 一、问题分析

### 1.1 问题背景

149 个有扰流窗口覆盖 D1–D8 八种扰流状态，分布在 20260608–20260615 共 8 天。扰流件导致速度剖面畸变（非对称流、旋流、偏流），使固定权重积分产生系统性偏差。需要识别扰流特征规律并建立补偿模型。

### 1.2 问题重述（四问）

| 子问题 | 核心任务 |
|--------|---------|
| (1) 特征选择 | 哪些特征对区分无扰流和有扰流最有用？ |
| (2) 扰流聚类 | D1–D8 可否归并为若干类？给出聚类依据 |
| (3) 在线识别 | 不读取 `disturbance_id`，构造扰流状态在线识别规则 |
| (4) 流量补偿 | 扰流下是否需对不同流量点采用不同补偿？ |

### 1.3 核心约束

- 离线分析可使用 `disturbance_id` 研究规律
- **在线规则不得依赖人工输入的扰流编号**
- 补偿规则和参数须在测试前固定

### 1.4 问题类型

特征工程 + 统计判别 + 聚类分析 + 分段补偿。核心是挖掘四类特征（profile/AB差异/dynamic/zero）对扰流的响应规律。

## 二、子问题(1)：扰流敏感特征识别

### 2.1 候选特征池

| 特征类别 | 具体变量 | 物理含义 |
|---------|---------|---------|
| 声道等效观测 | chord0–chord4 | 五声道等效流速贡献 |
| A/B方向差异 | ab0–ab4 | 同声道双方向不对称度 |
| 剖面特征 | profile_top_bottom, profile_center_all, profile_edge_inner, profile_inner_skew | 截面速度分布形状 |
| A/B汇总 | profile_ab_abs, profile_swirl | 方向差异总量 + 旋流代理 |
| 启停动态 | dyn_first_0p1_s, dyn_tail_0p1_s, dyn_start_over_plateau, dyn_end_over_plateau, dyn_plateau_cv, dyn_active_eq_s | 窗口稳定性 |
| 零点状态 | zero_rate_med, zero_rate_mad, zero_age_s | 零漂基线 |

### 2.2 判别力评估方法

对每个特征 $x$，用两个互补指标衡量 D0 vs D1–D8 的区分能力：

**效应量（Cohen's d）**：衡量两类均值差异的标准化幅度。

$$d = \frac{\bar{x}_D - \bar{x}_{D0}}{\sqrt{(s_D^2 + s_{D0}^2)/2}}$$

其中 $\bar{x}_D, s_D$ 为有扰流（D1–D8）的均值和标准差；$\bar{x}_{D0}, s_{D0}$ 为无扰流（D0）。

| Cohen's d | 效应强度 |
|-----------|---------|
| d > 0.8 | 大效应 |
| 0.5 < d ≤ 0.8 | 中效应 |
| d ≤ 0.5 | 小效应 |

**AUC（ROC曲线下面积）**：以特征值为判别分数，计算区分 D0 vs D1–D8 的 AUC。AUC ∈ [0.5, 1.0]，越接近 1 判别力越强。

### 2.3 数据分析结果

基于 159 窗口实际数据：

| 特征 | D0 均值±SD | D1–D8 均值±SD | Cohen's d | AUC（预期） | 判别力 |
|------|-----------|--------------|-----------|------------|--------|
| **profile_ab_abs** | 0.0115±0.0059 | 0.2101±0.0060 | **33.1** | ≈1.00 | ★★★★★ |
| **profile_swirl** | -0.0060±0.0037 | 0.1230±0.0028 | **39.6** | ≈1.00 | ★★★★★ |
| profile_edge_inner | 0.9258±0.0024 | 0.9503±0.0108 | 3.0 | ≈0.99 | ★★★★ |
| profile_center_all | 1.0387±0.0016 | 1.0256±0.0060 | 2.8 | ≈0.98 | ★★★★ |
| zero_rate_med | 0.8351±0.0633 | 1.7475±0.6686 | 1.9 | ≈0.90 | ★★★ |
| zero_age_s | 272.8±264.0 | 865.3±606.6 | 1.3 | ≈0.80 | ★★ |
| zero_rate_mad | 0.1227±0.0328 | 0.2837±0.3105 | 0.7 | ≈0.70 | ★★ |
| profile_top_bottom | -0.0118±0.0039 | -0.0293±0.0508 | 0.5 | ≈0.65 | ★★ |

### 2.4 结论

**最强判别特征**：`profile_ab_abs`（d=33.1）和 `profile_swirl`（d=39.6）。两者在 D0 中接近零，D1–D8 中显著偏离零，D1–D8 组内 CV 仅约 3%，说明扰流状态下这两个量是稳定可测的物理信号而非噪声。物理含义明确：扰流件必然引入方向不对称和旋流，适合作为在线检测的核心指标。

**次级判别特征**：`profile_edge_inner`（d=3.0）和 `profile_center_all`（d=2.8）也表现出显著差异，可作为辅助特征用于更细粒度的扰流类型判别。

**动态和零点特征**判别力较弱，但可作为有效性门和冷启动处理的依据。

## 三、子问题(2)：扰流状态聚类

### 3.1 聚类特征

取 10 维特征向量 $z$，包含：
- 归一化声道贡献 $\tilde{c}_j = c_j\,/\!\sum_k c_k$（$j=0,\ldots,4$），消除流量幅度影响
- A/B 方向差异 ab0–ab4
- 剖面汇总特征 `profile_ab_abs` 和 `profile_swirl`

**预处理（关键步骤）**：

1. **Z-score 标准化**：各维度减去 D0 均值、除以 D0 标准差。此举以无扰流状态为"零参考点"，使扰流的偏离在各维度上可比：
   $$z_j^* = \frac{z_j - \bar{z}_j^{(D0)}}{\sigma_j^{(D0)}}$$

2. **PCA 降维**：10 维特征仅 8 个样本（8 种扰流），直接聚类面临维度灾难。先对 D1–D8 的窗口级特征矩阵做 PCA，保留累计方差 ≥ 90% 的主成分（预期 2–4 个），再在主成分空间聚类。

### 3.2 方法：层次聚类 + 轮廓系数

**步骤**：
1. 对每个 D1–D8 扰流状态，取该状态下所有窗口的 PCA 降维后特征的均值作为该扰流的代表向量
2. 计算 8×8 欧氏距离矩阵和 Mahalanobis 距离矩阵
3. 用 Ward 法做层次聚类
4. 以轮廓系数确定最优聚类数 $K \in \{2,3,4\}$

**判别依据**：
- 类内紧致：同类的扰流状态剖面畸变模式相似
- 类间分离：不同类的扰流状态剖面畸变模式有本质差异
- 物理可解释：每类对应一种扰流机理（弯头效应、半板阻挡、缩径等）

### 3.3 D1–D8 独立分布分析

对每种扰流状态，分析其特征分布特征：

| 扰流 | 天数 | 窗口数 | 流量点覆盖 | 分析要点 |
|------|------|--------|-----------|---------|
| D1 | 2天(14,15) | 21 | 20–80 | 跨日稳定性？ |
| D2 | 1天(15) | 23 | 20–80 | 单日样本最多 |
| D3 | 2天(13,14) | 21 | 20–80 | 跨日一致性？ |
| D4 | 1天(13) | 22 | 20–70 | 缺少 FP=80 |
| D5 | 1天(09) | 17 | 20–70 | 缺少 FP=80 |
| D6 | 1天(10) | 14 | 20–80 | 样本最少 |
| D7 | 1天(08) | 13 | 30–80 | 缺少 FP=20 |
| D8 | 1天(12) | 18 | 20–80 | — |

**关键观察**：D1 和 D3 各有 2 天数据，可检验同一扰流在不同日期的特征稳定性。若同扰流跨日特征差异小于跨扰流差异，则聚类仅需剖面信息即可，否则需考虑日期效应。

### 3.4 预期结果

预期 8 种扰流可归并为 2–4 类：

| 聚类 | 可能机理 | 剖面特征 |
|------|---------|---------|
| 类型A | 对称收缩/扩张 | chord1–chord3 均匀偏离 |
| 类型B | 单侧扰流（弯头类） | chord0/chord1 与 chord3/chord4 不对称 |
| 类型C | 旋流主导 | profile_swirl 和 profile_ab_abs 均大 |
| 类型D | 混合型 | 多特征同时异常 |

## 四、子问题(3)：扰流在线识别规则

### 4.1 问题约束

在线运行时不能读取 `disturbance_id`，只能用窗口级在线可计算特征判断扰流存在性及类型。

### 4.2 双层识别架构

```
窗口特征（在线可计算）
    │
    ▼
┌──────────────────────────┐
│ 第1层：扰流检测（二分类）  │
│ 输入：profile_ab_abs,    │
│       profile_swirl      │
│ 规则：双阈值 OR 判定      │
│ 阈值：D0 max + 3σ（固定） │
└──────┬───────────────────┘
       │ flag = 1（有扰流）
       ▼
┌──────────────────────────┐
│ 第2层：扰流分类（多分类）  │
│ 输入：PCA降维后的10维向量  │
│ 方法：Mahalanobis 距离    │
│       到最近聚类中心      │
│ 中心：离线固定（≤4类）    │
└──────┬───────────────────┘
       │
       ▼
   对应补偿策略
```

### 4.3 第1层：扰流检测规则

记 $p_{\text{abs}}$ = `profile_ab_abs`，$p_{\text{swirl}}$ = `profile_swirl`。

**双指标联合判定**：

$$\text{flag} = \begin{cases} 1, & |p_{\text{abs}}| > \tau_1 \;\text{OR}\; |p_{\text{swirl}}| > \tau_2 \\[4pt] 0, & \text{otherwise} \end{cases}$$

使用 OR（而非 AND）的理由：某些扰流可能仅产生旋流而不破坏对称性（或反之），OR 条件保证不漏检。

**阈值设定**：以 D0 的 max 值为基准加 $3\sigma$ 余量：

$$\tau_1 = \max_{i \in D0}|p_{\text{abs},i}| + 3\sigma_{D0} = 0.0177 + 3 \times 0.0059 = 0.035$$

$$\tau_2 = \max_{i \in D0}|p_{\text{swirl},i}| + 3\sigma_{D0} = 0.0097 + 3 \times 0.0037 = 0.021$$

**误判风险分析**：

| 误判类型 | 风险 | 后果 |
|---------|------|------|
| D0 判为扰流（假阳性） | 极低（3σ 阈值） | D0 窗口被错误补偿，轻微增大 D0 误差 |
| 扰流判为 D0（假阴性） | 极低（3σ 阈值） | 扰流不补偿，$u_{\text{nor},d}$ 可能大幅超标 |

因 d > 33，阈值在两分布之间有 >30σ 的间隔，两类误判概率均接近零。

### 4.4 第2层：扰流类型判别

以 PCA 降维后的向量代替原始 10 维向量，使用 Mahalanobis 距离（考虑类内协方差）替代欧氏距离：

$$d_M(z, \mu_c) = \sqrt{(z - \mu_c)^\top \Sigma_c^{-1} (z - \mu_c)}$$

新窗口归入距离最近的聚类中心所属类型。$\mu_c$ 和 $\Sigma_c$ 由对应类的窗口样本离线估计。

用 Mahalanobis 而非欧氏距离的理由：不同扰流类型的特征分布可能具有不同的协方差结构（如某一类 AB 差异波动大、另一类剖面波动大），Mahalanobis 自动考虑尺度差异和相关性。

**避免过拟合**：仅用 ≤4 个聚类中心（而非 8 个扰流编号），大幅降低分类复杂度。每个类别的协方差矩阵需正则化（shrinkage）以防止小样本退化。

## 五、子问题(4)：分流量点扰流补偿

### 5.1 问题分析

扰流导致的剖面畸变在不同流量下表现不同：
- 低流量（20–30）：流速低，惯性效应弱，扰流影响更局部化
- 高流量（60–80）：流速高，湍流混合强，扰流影响被部分"洗匀"

需判断是否需分流量点补偿。

### 5.2 分析框架

对每种扰流类型（聚类后的类别），在 7 个流量点上分别计算扰流引入的额外偏差：

$$\Delta_{p}^{(c)} = \bar{e}_{p,\text{dist}}^{(c)} - \bar{e}_{p,\text{ref}}$$

其中 $\bar{e}_{p,\text{ref}}$ 为问题2修正后的无扰流流量点误差（预期接近零）。

若 $\Delta_{p}^{(c)}$ 在不同 $p$ 之间差异显著（ANOVA p < 0.05），则该扰流类型需分流量点补偿。

### 5.3 补偿模型

**形式一：加性补偿**（推荐，默认方案）

$$V_{\text{final}} = V_{\text{base}} \cdot \bigl(1 + \beta_p + \delta_{p}^{(c)}\bigr)$$

其中 $\beta_p$ 来自问题2的无扰流流量点修正，$\delta_{p}^{(c)}$ 为扰流类型 $c$ 在流量点 $p$ 的额外修正：

$$\delta_{p}^{(c)} = \frac{1}{N_{p}^{(c)}} \sum_{i \in \Omega_{p}^{(c)}} \frac{V_i^{\text{std}} - V_i^{\text{base}}}{V_i^{\text{base}}} - \beta_p$$

**形式二：乘性补偿**（备选，适用于 ANOVA 不显著时）

$$V_{\text{final}} = V_{\text{base}} \cdot (1 + \beta_p) \cdot (1 + \gamma^{(c)})$$

仅一个全局扰流因子 $\gamma^{(c)}$，由该类型所有流量点的平均偏差估计。

### 5.4 补偿策略选择流程

```
对每种扰流类型 c:
  │
  ├─ ANOVA 检验 Δ_p^(c) 跨流量点是否一致
  │
  ├─ p ≥ 0.05（无显著差异）
  │   → 全局乘性补偿 γ^(c)（1个参数）
  │   → 理由：扰流影响与流量无关，节省参数
  │
  └─ p < 0.05（有显著差异）
      → 分流量点加性补偿 δ_p^(c)（≤7个参数）
      → 理由：扰流影响随流量变化，须分别修正
```

### 5.5 小样本流量点的补偿策略

某些扰流×流量点组合可能仅有 1–2 个窗口（如 D7 仅 13 窗口 / 5 流量点 = 平均 2.6 窗/点）。对于 $N_p^{(c)} < 3$ 的组合：

1. 使用同扰流类型内相邻流量点的 $\delta$ 值做线性插值
2. 或退化为该类型的全局均值，并标记为低置信度

## 八、注意事项

1. 聚类前必须做 Z-score 标准化和 PCA 降维——10维特征对8个样本直接聚类的结果不可靠
2. D0 仅 10 窗口，Cohen's d 的标准误 ≈ $\sqrt{4/n} \approx 0.63$（大样本近似），应汇报 95% CI
3. Mahalanobis 距离需做协方差 shrinkage——小类样本（如仅2天、每类仅几窗口）下样本协方差退化
4. 补偿参数不得对单个窗口调优——必须按组（扰流类型×流量点）聚合
5. 在线识别规则中的阈值须在论文中给出完整物理推导，强调其与声学原理的关联
6. `profile_ab_abs` 和 `profile_swirl` 的判别力极强（d > 30），是本题最重要的物理发现之一，论文中应重点论述
7. D1 和 D3 的跨日数据是检验同扰流稳定性的天然实验——须在论文中报告

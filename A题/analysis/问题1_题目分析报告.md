# 问题1：超声波流量计基础模型 — 建模分析

## 一、问题分析

### 1.1 问题背景

超声波流量计通过斜向声道中顺逆流传播时间差估计流速。单声道仅能测量声道线平均速度，而实际管道截面速度分布不均匀（层流呈抛物线、湍流呈指数型），需多声道积分逼近面平均流速。

本题数据来自一台五声道超声波流量计，包含159个窗口的等效声道观测量（chord0–chord4）、A/B方向差异、剖面/动态/零点特征，以及标准装置体积参考值。

### 1.2 问题重述

问题1要求：
1. 从时差法原理出发，推导单声道流速估计公式，并说明如何由流速计算体积流量
2. 解释多声道积分的必要性（速度剖面不均匀）
3. 给出一种多声道积分方法（如五声道加权积分），解释权重物理含义
4. 利用附件6/7构造单声道或多声道积分估计，或直接使用附件1中四套基础估计量（phys6/owics/lagrange/equal_weight），比较各方法在不同流量点和工况下的误差

### 1.3 问题类型

物理模型推导 + 数值积分 + 误差统计分析。核心是理解时差法原理并评估不同积分方案的精度。

## 二、模型建立

### 2.1 单声道流速公式推导

设声道长度 $L$，与管轴夹角 $\theta$，水中声速 $c$，轴向流速 $v$。声波在管壁固定探头间传播，需叠加水的拖曳：

$$t_+ = \frac{L}{c + v\cos\theta}, \qquad t_- = \frac{L}{c - v\cos\theta}$$

**消 $c$ 推流速**。$\Delta t = t_- - t_+$，通分得：

$$\Delta t = \frac{2Lv\cos\theta}{c^2 - v^2\cos^2\theta} \tag{1}$$

$$t_+t_- = \frac{L^2}{c^2 - v^2\cos^2\theta} \tag{2}$$

(1) ÷ (2)，分母精确消去：

$$v = \frac{L}{2\cos\theta} \cdot \frac{\Delta t}{t_+ \cdot t_-}$$

本题等效 $\cos\theta = 1/\sqrt{2}$（附件7）。

**体积**：窗口持续 $T$ 秒，等效截面积 $A = 0.13138$ m²，$V = A \cdot v \cdot T$。

### 2.2 多声道积分的必要性

管道截面速度分布不均匀：
- 层流：$v(r) = v_{\max}(1 - r^2/R^2)$，中心速度为面平均的2倍
- 湍流：$v(r) = v_{\max}(1 - r/R)^{1/n}$，$n \approx 6\text{–}10$

单声道只能测量所在位置的线平均速度，无法代表面平均。多声道在截面不同高度布设声道，通过加权积分逼近面平均：
$$\bar{v} = \sum_{i=0}^{4} w_i \cdot v_i$$

### 2.3 多声道积分方法

**Gauss-Jacobi 积分**（本题 Phys6/OWICS 的实现基础）：

对于圆形管道，流量 $Q = \int_{-R}^{R} v(y) \cdot 2\sqrt{R^2 - y^2} \, dy$。将 $y$ 归一化为 $t = y/R$，权重函数 $W(t) = (1-t)^\alpha (1+t)^\beta$，取 $\alpha = \beta = 1/2$。高斯求积节点对应声道位置 $t_i$，求积系数即为权重 $w_i$。

附件7给出五套权重：

| 方案 | 特征 | 权重和 |
|------|------|--------|
| Phys6 | 工程法原始系数 | ≈0.993（不归一） |
| OWICS | 归一化声道位置积分 | 1.0，与 Gauss-Jacobi 一致 |
| Gauss-Jacobi | 标准圆管数值积分 | 1.0 |
| Lagrange | Lagrange 插值型积分 | 1.0 |
| 等权 | 简单算术平均 | 1.0 |

各声道权重对应其代表截面积占总截面积的比例。边缘声道（chord0/chord4，$|y| = 0.727R$）权重最大（~0.22），中心声道（chord2，$y=0$）权重居中（~0.33），中间声道（chord1/chord3，$|y|=0.266R$）权重最小（~0.11）。这符合圆管截面积分布：靠近管壁的环带面积大，权重应大。

## 三、误差对比方案

### 3.1 利用附件1已有估计量

附件1已提供四套基础积分的结果列，可直接做误差统计：

| 列名 | 方法 |
|------|------|
| phys6_volume_m3 | 五声道固定物理权重 |
| owics_volume_m3 | OWICS归一化权重 |
| lagrange_volume_m3 | Lagrange型积分 |
| equal_weight_volume_m3 | 五声道等权 |

每行对应一个窗口，`standard_volume_m3` 为真值。相对误差 $e_i = (\hat{V}_i - V_i^{\text{std}}) / V_i^{\text{std}} \times 100\%$。

### 3.2 评价指标

| 指标 | 含义 | 计算 |
|------|------|------|
| MAE | 平均绝对误差 | $\frac{1}{n}\sum |e_i|$ |
| 均值误差 | 系统偏差 | $\frac{1}{n}\sum e_i$ |
| 最大绝对误差 | 最差案例 | $\max |e_i|$ |
| 分组重复性 SD | 同日同流量组内标准差 | $\sqrt{\frac{1}{n_g-1}\sum(e_{g,k} - \bar{e}_g)^2}$ |

### 3.3 对比维度

1. 全窗口 MAE、均值误差、最大误差
2. 按流量点（20–80）分层统计
3. 按扰流/无扰流分层统计
4. 按日期分组重复性

## 六、注意事项

1. attachment4_baseline_summary.csv 已有四方法的baseline统计（MAE、均值误差、最大误差），代码结果应与之一致，用于交叉验证
2. 等权方法（equal_weight）预期误差最大——中心声道不区分权重，物理上不合理
3. Phys6 权重和 ≈0.993，会产生约0.7%的系统低估；均值误差约 -0.136%，与附件4一致
4. OWICS 和等权均存在系统高估（均值误差分别0.506%和0.637%），可能是归一化导致
5. Lagrange MAE 最小（0.184%），但均值误差接近零（-0.008%），表现最均衡

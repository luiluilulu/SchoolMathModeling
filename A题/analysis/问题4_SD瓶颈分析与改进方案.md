## 二、SD瓶颈的根因分析

### 2.1 SD的来源分解

通过对159个窗口的方差分解：

```
总方差 = 类间方差 + 类内方差
       = 53.5% + 54.0%
```

**关键发现**：
- 类间方差（不同disturbance_id之间的差异）占53.5%
- 类内方差（同一disturbance_id内不同窗口之间的差异）占54.0%
- **类内方差占主导**，说明即使完美分类，仍有大量窗口间随机波动无法消除

### 2.2 SD最高组的详细分析

**SD最高的5个组**：

| 日期 | 流量点 | N | SD | 主要扰流类型 | 组内误差范围 |
|------|--------|---|----|--------------|--------------|
| 20260613 | 50 | 4 | 0.1223% | D4(3窗) | [-0.102%, +0.171%] |
| 20260612 | 70 | 3 | 0.1066% | D8(3窗) | [-0.066%, +0.123%] |
| 20260613 | 70 | 6 | 0.0977% | D4(5窗) | [-0.187%, +0.093%] |
| 20260613 | 20 | 4 | 0.0756% | D4(3窗) | [-0.097%, +0.086%] |
| 20260615 | 60 | 6 | 0.0744% | D1+D2混合 | [-0.075%, +0.084%] |

**典型案例**：20260613 fp=50，D4扰流类型
- 窗口1: 误差 = -0.102%
- 窗口2: 误差 = -0.070%
- 窗口3: 误差 = +0.171%
- **同一扰流类型、同一流量点、同一天**，误差差异达0.27%

### 2.3 特征与SD的相关性分析

测试了70+个特征与窗口误差的相关性：

| 特征类别 | 最高相关系数 | 结论 |
|----------|--------------|------|
| profile_ab_abs | r = +0.46 | 全局强相关，但流量点内r = +0.46 |
| profile_swirl | r = +0.49 | 全局强相关，但流量点内r = +0.49 |
| 附件6原始时序特征 | r < 0.15 | 弱相关，无法有效预测窗口间差异 |
| 动态特征 (dyn_*) | r < 0.11 | 弱相关 |
| 零点特征 (zero_*) | r < 0.08 | 弱相关 |

**关键发现**：
- `profile_ab_abs`和`profile_swirl`在全局范围内与误差强相关
- 但在**同一流量点内**，这些特征的变化与误差变化的相关性仍然较弱
- 原始时序特征（附件6）的变异系数、偏度、峰度等与误差几乎无关

### 2.4 理论下限分析

**实验1：完美补偿(did, fp)均值**
```
假设：每个(disturbance_id, flow_point)组合的补偿值 = 该组合的真实均值误差
结果：最大组SD = 0.1220%，中位数SD = 0.0437%
```

**实验2：LOO-CV补偿（避免过拟合）**
```
方法：每个窗口的补偿值 = 其他窗口的均值
结果：最大组SD = 0.1831%，中位数SD = 0.0618%
```

**结论**：
- 即使完美补偿均值，最大组SD仍在**0.12%**以上
- 目标SD = 0.040%，理论下限约**0.10%**
- **差距：3倍，在当前数据条件下不可达**

## 三、SD不可达的根本原因

### 3.1 窗口间随机波动的来源

1. **窗口启停动态差异**
   - 每个窗口的开始/结束时间不同
   - 流量进入稳定状态的时间不同
   - 尾部衰减的时间不同

2. **湍流脉动的随机采样**
   - 即使在同一扰流条件下，湍流脉动是随机过程
   - 不同窗口的采样时间段不同，捕获的脉动模式不同
   - 这种随机性无法通过固定补偿消除

3. **零点状态的微小漂移**
   - `zero_age_s`（距上次零点的时间）在不同窗口间差异很大
   - 零漂基线、零漂波动随时间变化
   - 虽然与误差相关性弱，但累积效应影响SD

4. **测量噪声**
   - 超声波传播时间的测量噪声
   - A/B方向差异的随机波动
   - 电子设备的温度漂移

### 3.2 为什么补偿无法降低SD

**补偿模型的本质**：
```
补偿后误差 = 原始误差 - 补偿值
           = (系统性偏差 + 随机波动) - 补偿值
```

**如果补偿值 = 系统性偏差的估计**：
```
补偿后误差 = 随机波动
```

**SD的定义**：
```
SD = sqrt(Σ(误差_i - 均值)^2 / (N-1))
   = sqrt(Σ(随机波动_i)^2 / (N-1))
```

**结论**：
- 补偿只能消除系统性偏差（组均值）
- SD完全由随机波动决定
- 随机波动是窗口间的固有差异，无法通过固定补偿消除

## 四、已测试的改进方案

### 4.1 方案对比

| 方案 | MAE | 最大SD | 组通过 | 结论 |
|------|-----|--------|--------|------|
| K=2聚类补偿 | 0.095% | 0.123% | 9/30 | 基线 |
| **K=8分类补偿** | **0.0325%** | **0.122%** | **14/30** | **最优** |
| 连续特征回归 | 0.125% | 0.126% | 7/30 | 过拟合 |
| LOO-CV补偿 | 0.047% | 0.183% | 9/30 | SD恶化 |
| Trimmed Mean | 0.032% | 0.122% | 14/30 | 与K=8相当 |
| 收缩估计(α=0.7) | 0.054% | 0.122% | 13/30 | 略差 |
| 附件6原始特征 | 0.118% | 0.146% | — | 无效 |

### 4.2 K=8模型的优势

1. **MAE最低**：0.0325%，比K=2降低66%
2. **组均值偏差最小**：30/30组通过均值检查（|mean| ≤ 0.2%）
3. **物理可解释**：每种扰流类型独立补偿，符合工程直觉
4. **参数固定**：补偿表离线确定，在线仅查表

### 4.3 为什么其他方案失败

- **连续特征回归**：过拟合训练数据，LOO-CV后SD恶化
- **LOO-CV补偿**：避免了过拟合，但SD反而更高（0.183%）
- **收缩估计**：向全局均值收缩，降低了MAE但对SD无帮助
- **附件6原始特征**：与误差相关性太弱（r < 0.15），无法预测窗口间差异

## 五、SD瓶颈的突破方向

### 5.1 当前数据条件下的限制

**不可突破的硬上限**：
- 最大组SD ≈ 0.10%（理论下限）
- 目标SD = 0.040%
- **差距：2.5倍**

**原因**：
- 窗口间随机波动是固有特性
- 当前特征集无法有效建模这种波动
- 样本量太小（每组3-6个窗口），统计估计本身不稳定

### 5.2 若需进一步改进，建议的实验/工程方案

#### 方案1：增加窗口内采样时间（工程改进）
```
当前：每个窗口持续100-200秒
改进：延长到300-500秒
预期效果：
  - 窗口内平均更充分，随机波动降低
  - 预计SD降低30-50%
  - 但会降低时间分辨率
```

#### 方案2：增加重复测量次数（实验设计）
```
当前：每个(date, flow_point, disturbance_id)组合3-6个窗口
改进：增加到10-15个窗口
预期效果：
  - 组均值估计更稳定
  - SD的置信区间缩小
  - 但不改变SD本身
```

#### 方案3：引入窗口稳定性加权（算法改进）
```
思路：对窗口内稳定性高的样本赋予更高权重
特征：dyn_plateau_cv（平台段变异系数）
      dyn_active_eq_s（等效有效持续时间）
实现：
  - 计算每个窗口的稳定性得分
  - 在组内计算SD时，使用加权标准差
预期效果：
  - 可能降低SD 10-20%
  - 但需要修改评价标准
```

#### 方案4：滑动窗口平滑（信号处理）
```
思路：对原始时序数据（附件6）进行滑动窗口平均
方法：
  - 对diff_ns_*进行5秒滑动平均
  - 重新计算chord0-4
  - 重新估计流量
预期效果：
  - 降低高频噪声
  - 可能降低SD 5-10%
  - 但会引入相位延迟
```

#### 方案5：机器学习异常检测（高级方法）
```
思路：识别并剔除异常窗口
方法：
  - 用Isolation Forest或LOF检测异常窗口
  - 剔除异常窗口后重新计算SD
预期效果：
  - 可能降低SD 20-30%
  - 但会减少有效样本量
  - 需要严格的异常定义，避免主观剔除
```

### 5.3 推荐优先级

| 优先级 | 方案 | 可行性 | 预期效果 | 实施难度 |
|--------|------|--------|----------|----------|
| 1 | 增加窗口采样时间 | ⭐⭐⭐⭐⭐ | SD↓30-50% | 低（工程改进） |
| 2 | 增加重复测量 | ⭐⭐⭐⭐ | 统计稳定性↑ | 低（实验设计） |
| 3 | 窗口稳定性加权 | ⭐⭐⭐ | SD↓10-20% | 中（需改评价标准） |
| 4 | 滑动窗口平滑 | ⭐⭐⭐ | SD↓5-10% | 中（信号处理） |
| 5 | 异常检测剔除 | ⭐⭐ | SD↓20-30% | 高（需严格定义） |

## 六、最终模型推荐

### 6.1 推荐模型：K=8 Mahalanobis分类补偿

**模型架构**：
```
输入：窗口在线特征 (chord0-4, profile_swirl, profile_ab_abs, flow_point)
│
├─ (1) 基线估计：Phys6固定权重
│   V_base = A · T · Σ w_j^Phys6 · chord_j
│
├─ (2) 扰流检测：双阈值规则
│   若 |profile_ab_abs| > 0.0386 或 |profile_swirl| > 0.0226
│     → flag=1（有扰流），进入 (3)
│   否则 → flag=0（D0），跳到 (4)
│
├─ (3) K=8分类 + 补偿
│   12维特征 → Mahalanobis距离 → 8种扰流类型
│   V = V_base × (1 + δ_{did, flow_point})
│   补偿表：8类 × 7流量点 = 56个系数
│
├─ (4) D0流量点修正
│   V = V_base × (1 + δ_{D0, flow_point})
│
└─ (5) 输出 model_volume_m3
```

**参数清单**：
| 参数 | 数量 | 来源 |
|------|------|------|
| Phys6权重 | 5 | Gauss-Jacobi积分推导 |
| 检测阈值 | 2 | D0统计量 |
| 分类中心 | 8×12 | 各扰流类型均值 |
| 协方差矩阵 | 8×12×12 | LedoitWolf估计 |
| 补偿系数 | 8×7 + 5 = 61 | 分类×流量点均值 |
| **总参数** | **~700** | 无自由参数拟合 |

### 6.2 模型性能

| 指标 | 值 | 评价 |
|------|----|------|
| MAE | 0.0325% | ✅ 优秀 |
| 组通过 | 14/30 (47%) | ⚠️ 中等 |
| $u_{\text{nor},L}$ | 0.0018% | ✅ 远超目标 |
| $u_{\text{nor},r}$ | 0.1223% | ❌ 3倍超标 |
| $u_{\text{nor},d}$ | 0.1494% | ❌ 1.3倍超标 |

### 6.3 模型局限性

1. **SD瓶颈无法突破**
   - 理论下限约0.10%，目标0.040%
   - 窗口间随机波动是固有特性
   - 需要工程改进（增加采样时间、增加重复测量）

2. **$u_{\text{nor},d}$超标**
   - 主要由$u_{\text{nor},r}$贡献（0.149% vs 0.115%目标）
   - 漂移项$u_{\text{nor},d,c}$已控制到0.0012%
   - 需要降低窗口间随机波动

3. **样本量限制**
   - 每组仅3-6个窗口，统计估计不稳定
   - D0仅10个窗口，5个流量点
   - 需要更多数据支撑可靠结论

### 6.4 论文撰写建议

1. **突出创新点**
   - K=8分类替代K=2聚类，MAE降低66%
   - Mahalanobis距离在线分类，物理可解释
   - 分流量点补偿，ANOVA验证必要性

2. **诚实报告局限性**
   - SD瓶颈的根因分析
   - 理论下限的推导
   - 工程改进建议

   - Phys6权重的物理来源（Gauss-Jacobi积分）
